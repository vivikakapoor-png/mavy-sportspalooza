Mavy Trivia Web App - Starter
Files included:
- index.html
- style.css
- script.js
- questions.json (150 questions)
- banner.png (placeholder or included banner image)

How to use:
1. Unzip or copy these files into your repo folder.
2. Open index.html in a browser or push to GitHub Pages.
3. Choose a category (All/Seahawks/Mariners/General), enter a name, Start Quiz.
4. Quiz uses 15 random questions from the chosen pool. Results screen shows score.
5. Save to leaderboard to store top scores in the browser (localStorage).

Notes:
- Replace banner.png with your final banner image if desired.
- To edit/add questions, modify questions.json (keep same object shape).
